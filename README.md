# Grand Auto Services Website

Premium automotive service landing page built with Next.js, Tailwind CSS, and Framer Motion.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```

## Features

- Dark luxury theme with glassmorphism styling
- Smooth animations using Framer Motion
- Mobile responsive design
- Premium automotive service sections for Grand Auto Services
